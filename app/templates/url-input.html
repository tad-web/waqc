This will check the page at the URL submitted below and a randomly selected subpage for the following potential accessibility violations:

  • Skip link check: We note if there is an html link that matches an example skip link label from our list of good skip link labels . Unfortunately, we cannot find skip links contained in javascript.

  • Image alt text: We note if an image has no alt text or if the alt text contains a file extension.
  • Header hierarchy: We note if the first header on a page is not an h1 or if header levels are skipped.
  • Form labels: We note if an editable form in a field is unlabeled.
